Ajuda do ScriptFlow

ScriptFlow é uma plataforma para criação de roteiros cinematográficos e storyboards.

Como usar:

  • Use os botões na barra de ferramentas para inserir elementos de roteiro
  • Clique em qualquer elemento para editá-lo
  • Salve seu roteiro localmente no seu navegador
  • Exporte para FDX (Final Draft) ou PDF
  • Na aba Storyboard, crie visualizações para suas cenas

Atalhos:

  • Ctrl+S: Salvar roteiro
  • Ctrl+O: Abrir roteiro
  • Ctrl+P: Exportar para PDF
