Como colar seu código HTML no GitHub

Copie o Código HTML

Primeiro, copie todo o código HTML que eu forneci anteriormente:

// TODO: Todo o código HTML será mostrado aqui // Este é um espaço reservado para o código completo <!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <title>ScriptFlow</title> <style> body { font-family: Arial, sans-serif; } </style> </head> <body> <h1>Minha Plataforma</h1> <p>Conteúdo da plataforma...</p> </body> </html>
Código copiado para a área de transferência!

Dica: Clique no botão acima para copiar automaticamente todo o código HTML.

Cole no GitHub

Depois de copiar o código, siga estes passos para colá-lo no GitHub:

1

No GitHub, clique em "Add file" → "Create new file"

Isso abrirá o editor de arquivos do GitHub.

2

Nomeie o arquivo como index.html

No campo "Name your file", digite index.html

3

Cole o código na área grande de texto

Clique na área "Enter file contents here" e cole o código (Ctrl+V ou Command+V).

// Cole seu código HTML aqui
4

Adicione uma mensagem de commit

No campo "Commit new file", digite uma mensagem como "Adicionar arquivo inicial".

5

Clique em "Commit new file"

Isso salvará o arquivo no seu repositório.

Verifique se deu certo

Depois de commitar o arquivo, verifique se tudo funcionou corretamente:

1

Volte para a página principal do repositório

Você deve ver o arquivo index.html na lista de arquivos.

2

Clique no arquivo para visualizá-lo

Verifique se o conteúdo está correto.

3

Ative o GitHub Pages

Vá em Settings → Pages → Selecione a branch "main" → Clique em Save.

4

Acesse seu site

Em alguns minutos, seu site estará disponível em: https://seuusuario.github.io/nome-do-repositorio

Abrir GitHub

Se encontrar qualquer problema, recarregue esta página e tente novamente. O código HTML estará disponível para copiar novamente.
